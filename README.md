# Web Game 03

A multiplayer web game built with React, Vite, and Socket.io. Features a vibrant, Neo-Brutalism UI style adapted from HiveMindCursor.

## Tech Stack
- **Frontend:** React + Vite
- **Real-time Communication:** Socket.io (Client ready)
- **Icons:** Lucide React
- **Styling:** Custom CSS with Neo-Brutalism elements

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.
